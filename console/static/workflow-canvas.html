GEM² + Lobster Trap: Two-Layer AI Governance

Enterprise AI needs two trust layers — security and truth. Lobster Trap blocks prompt-layer attacks at the wire in <15 ms. GEM² audits reasoning-layer truth after the wire. Together: a four-gate governance pipeline that security and compliance teams can read.

🦞
Lobster Trap — L0 (ingress) + L3 (egress) "Is this safe?" — fast, deterministic DPI. L0 scans the NL prompt at ingress; L3 scans the rendered output at egress. Regex + 22 metadata fields + YAML policy. Zero runtime cost.
🔬
GEM² Truth Filter — L1 (P-check) + L2 (O-check) "Is this true?" — semantic + epistemic evaluation. L1 validates input against type A & preconditions; L2 validates output against type B & postconditions. TPMN formal-methods scoring (0-100).
🤖
